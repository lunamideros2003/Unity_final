# Build del Juego – Instrucciones y Controles

Este repositorio contiene el proyecto de Unity y la build ejecutable en releases.  

---

## Cómo descargar la build

1. Ve a la sección **Releases** del repositorio.
2. Descarga el archivo correspondiente
3. Extrae el contenido del archivo winrar comprimido.
4. Ejecuta el archivo principal del juego fps_tal.exe

---
